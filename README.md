# Extract-Images
### Pre-requisitos:
* nodejs & mongodb

### Instalação:
```ssh
  $ git clone https://github.com/dg-jacquard/extract-images.git
  $ cd extract-images
  $ npm install
  ```
#### Development
```ssh
  $ npm i -g nodemon
  $ npm install
  $ nodemon bin/www
  ```
#### Production
```ssh
  $ npm i -g pm2
  $ npm install
  $ pm2 start bin/www
  ```
